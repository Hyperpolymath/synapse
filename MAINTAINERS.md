## Becoming a Maintainer

### Expert Contributor Path

1. Make 10+ accepted contributions
2. Demonstrate expertise in Zig, Rust, or Swift
3. Receive nomination from Core maintainer
4. Receive majority Core approval

### Core Maintainer Path

1. Serve as Expert contributor for 12+ months
2. Make sustained, high-quality contributions
3. Receive nomination from Core maintainer
4. Receive unanimous Core approval

## Responsibilities

### All Maintainers

- Follow the Code of Conduct
- Respond to issues and MRs in reasonable time
- Document decisions
- Mentor new contributors

### Core Maintainers Additionally

- Final merge authority on `main`
- Security vulnerability response
- Release management
- Governance decisions
